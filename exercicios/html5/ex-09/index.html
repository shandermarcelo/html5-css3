 Acima temos um elemento <ol>, sigla para ordered list. Esse tipo de lista indica que a ordem do seu conteudo possui importância, isso é representado através dos marcadores, que no caso são os números a frente do conteudo do elemento falando sobre o conteudo que <ol> contêm esses são adicionados através do elemento <li> ai você já tem uma ideia geral que como é a estrutura de uma lista, um elemento contem toda a estrutura e elementos de conteudo componhem essa estrutura.
 Agora vamos falar sobre alguns parametros que uma lista pode possuir. Como acima nós temos uma lista ordenada vamos falar sobre o paramtro type="" esse parametro é onde se define qual sera o tipo do marcador que a lista vai utilizar e existem 5 valores declaraveis veja um exemplo abaixo:

  1. "1" é o valor padrão. Marcador numérico

  2. "A" é o marcador de letras alfabéticas maiusculas

  3. "a" é o marcador de letras alfabéticas minusculas

  4. "I" é o marcador de numeros romanos maiusculos
  5. "i" é o marcador de numeros romanos minusculos

 Agora vamos falar sobre o parametro start="" esse é um parametro númerico que podemos usar para demarcar apartir de qual marcador uma lista deve começar. esse valor deve ser compativel com o valor declarado no parametro type="" e deve sempre ser númerico mesmo que o marcador utilizado seja alfanumerico


Listas Desordenadas

 Também é possivel desenvolver listas onde a ordem de seu conteudo não seja importante <ul> é o elemento que usamos para desenvolver uma unordered list. A funcionalidade da estrutura desse elemento é identica a de um elemento <ol> com <ul> contendo todo o conteudo, que é composto por elementos <li> porém, assim como o proposito desse tipo de lista difere de uma oredered list os parametros dela também são diferentes. primeiro vamos falar do type="" que assim como em uma oredered list, é onde se define qual sera o tipo do marcador do conteudo da lista. e nós temos três opções como valores possiveis.SABIA QUE O PARAMETRO start="" NÃO TEM UTILIDADE NESSE TIPO DE LISTA PORQUE O PROPOSITO DELA É A FALTA DE ORDENAMENTO E COMO OS MARCADORES SÃO SIMBOLOS IGUAIS NÃO É POSSIVEL DEMARCAR DE ONDE ALGO COMEÇA OU TERMINA. UMA SOLUÇÃO PARA ISSO É INTRODUZIR O CONTEUDO QUE SE DESEJA COMO PRIMEIRO EM PRIMERIO

  • "disc" é o valor para o marcador padrão. um circulo escuro

  • "square" é o valor para o marcador quadrado escuro

  • "circle" é o valor para o marcador cirular sem preenchimento

 Acima nós temos uma <dl>, que é uma descriptive list isso que dizer que esse tipo de lista possui um funcionamento diferente das <ol> e das <ul>, Aqui cada item sera composto por dois elementos <dt> que é Descritive term é onde se vai colocar o termo do item que vai pretencer a esse tipo de lista e <dd> que é descriptive details, aqui é onde uma descrição sobre o conteudo que esta no elemeento descritive term será feita. portanto CADA ITEM QUE PERTECER A UMA DESCRIPTIVE SERA SERA COMPOSTO POR ESSES DOIS ELEMENTOS, UM CONTENDO O TERMO DO ITEM E O OUTRO CONTENDO A DESCRIÇÃO DO MESMO
