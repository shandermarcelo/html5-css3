Acima temos um exemplo de exibição de código, o elemento <code> é utilizando para representar seu conteudo como código em um texto, elementos são representado com o auxilio de duas HTML Entities sendo elas a Html Entity que representa o simbolo de Menor que e a a Html Entity que representa o simbolo de Maior que. Estas Entities são uteis por causa dos simbolo que elas representam, estes não são possiveis de exibir em forma de texto porque as teclas deles já são compreendidas como os comandos para marcação. Enfim o que deve ser entendido aqui é que o elemento <pre> deve ser utilizando como container (alguns chamam isso de Elemento Pai) do elemento <code> quando o código a ser exibido por grande devendo ao mono espaçamento que <code> não reconhece mas <pre> tenta reconhecer

Direcionamento de texto

Para manipular a direção de exibição de um texto utilize o elemento <bdo> e em seu parametro dir="" declare como valor qual direcionamento o texto tera "lto" quer dizer "left to right" e vai manipular a direção do texto da esquerda para a direita o conteudo daqui emn diante já esta contido em um elemento <bdo> com o valor declarado em seu parametro dir como "ltr" o que não mudou nada. Isso esse é o valor padrão. "rtl" quer dizer "right to left" e manipula a direção do texto da direita para esquerda assim, esse você pode ver de uma olhada
