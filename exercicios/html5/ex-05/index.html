Hierarquia de Headings(Titulos)

 Na Web, o conteudo textual é desenvolvido da seguinte forma, primeiro nós temos os titulos, também conhecido com headings, estes tem a função de dermarcar e distinguir um tópico de outro, como pode existir tópicos dentro de tópicos e sub tópicos dentro de tópicos ou sub tópicos dentro de sub topicos o uso dos elementos de heading pode ser multiplicado dependendo do tema abordado. Entenda agora como funciona a Hierarquia dos elementos heading e como ela ajuda na organização de cotneudo em uma página web.

Entendendo a Estrutura hierarquica

 A ordem hierarquica que governa nos elementos heading é decresente, ou seja quanto menor o valor numero do elemento heading, maior sera sua posição nessa hierarquia mas não entenda isso como "Se estes são os elementos mais importantes então eu só vou usar eles"Todo elemento possui seu valor, especialmente elementos que possuem semantica, e os elementos heading possuem isso portanto entenda apenas que independente da posição de um elemento nessa hierarquia seu uso pode ser aplciado sem problemas.
 Primeiro vem o elemento <h1> este é o elemento que esta no topo da hierarquia e o conteudo que ele conter será considerado como "titulo principal", esse elemento é utilizado para demarcar a tematica que uma página aborda
 O elemento <h2> se encontra em segundo lugar na hierarquia, este elemento é utilizado para demarcar topicos da tematica.
 O elemento <h3> esta em terceiro lugar na hierarquia, este elemento é usado para demarcar sub topicos. muito utilizado para aprofundar assuntos de um topico.
 Os elementos <h4><h5>e <h6> seguem o mesmo método de uso do elemento <h3> demarcado sub topicos de sub topicos, sendo que os elementos que possuem maior posição hierarquica são os recebem o uso de tais elementos PORÉM SERÁ MUITO DIFICIL DE FAZER USO DESTES ELEMENTOS PORQUE <h3> é onde a maioria dos desenvolvedores usa desses elementos headings, O que faz o uso dos três ultimos ser bastante pontual.mas a função destes se mantem a mesma com a de um elemento <h3> ser sub topicos de headings em posições maiores

Exemplo textual de como os headings são usados

<h1>titulo principal do assunto<h1/>
<p>conteudo textual<p/>
<h2>topico do assunto<h2/>
<p>conteudo textual do topico<p/>
<h3>titulo de sub topico do assunto<h3/>
<p>conteudo textual<p/>

 O exemplo acima demonstra um pouco do funcionamento desses elementos, eu coloquei os elementos heading com uma emfase e os deixei em negrito para que se note a diferença deles para o conteudo textual, que sempre acompanha tais elementos; E ai você pode ver a lógica de um tema que possui seu titulo principal e mas que é composto por outros elementos heading, QUE DIVIDEM O ASSUNTO EM TOPICOS, ESSE METODO DE DIVIDIR UM ASSUNTO EM TOPICOS É USADO PARA ORGANIZAR O RACIOCINEO DE QUEM DESENVOLVEU TAL CONTEUDO E TAMBÉM PARA MANTER OS LEITORES NA LEITURA DO TEXTO, JÁ QUE TEXTOS COM CONTEUDO TEXTUAL LONGO SÃO TEDIOSOS DE SE LER

Outras considerações

 O exemplo do uso dos elementos heading acima não teve repetição de elementos, o que é possivel mas não se engane É POSSIVEL E SE DEVE USAR MAIS DE UM ELEMENTO HEADING CASO SEJA NECESSARIO VEJA MAIS UM EXEMPLO
