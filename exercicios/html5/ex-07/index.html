Principais Formatações

Negrito/Destaque

Aqui temos a aplicação do negrito utilizando o elemento <b> o que é categorizado como Forma(Forma porque elementos não Semânticos desse tipo eram usados somente pelo efeito que traziam, eles não possuem valor por isso)


Aqui temos a aplicação do termo em destaque, como se posse um negrito que utiliza um elemento semântico(Os elementos semânticos além de possuirem uma forma, que é tudo o que componhe os elementos de Forma, também possuem valor semântico, que é o proposito do seu uso <strong> é um elemento semântico que representa forte importancia, destaque a seu conteudo)

Itálico/Enfase

Aqui temos a aplicado do italico utilizando o elemento <i>, que é um elemento não semântico


Aqui temo a aplicação de um conteudo em destaque utilizando o elemento <em> que é um elemento semântico(em dá enfase a seu conteudo, e gera um efeito que dá um destaque)

Texto Marcado

Aqui nos temos a aplicação de um conteudo marcado essa marcação foi feita utilziando o elemento <mark>, ela possui essa habilidade de gerar um efeito semelhante ao de um marcador de texto. Por padrão essa cor é amarela Mas para alterala use estilização em CSS seja ela Inline, Interna ou Externa

Texto Grande e Texto Pequeno

Neste paragrafo há um texto Grande NÃO USEI O ELEMENTO <big> DE PROPOSITO PORQUE ELE ESTA OBSOLETO e um Texto Pequenosabia que apesar o elemento <small> ainda ser reconhecido devido a sua maior importancia(como se fosse uma semantica)

Texto Deletado

Podemos usar o elemento <del> para marcar palavras em um texto e indicar que se elas forem lidas, devem ser desconsideradas. Como que se as mesmas foram deletadas do texto assim

Texto Inserido

Podemos usar o elemento <ins> para marcar um conteudo textual como inserido para dar uma enfase e indicar que ele foi adicionado depois desse jeito


também existe o elemento <u> que não é semântico, este elemento tinha a função de gerar um efeito de sublinhar seu conteudo. Use o elemento <ins> para se quiser sublinhar sem estilizações CSS mas sabia que este elemento possui valor semântico de informar que seu conteudo é texto que foi adicionado depois o seu sublinhar é apenas um efeito que da destaque

Conteudo Sobescrito

todo conteudo que se deseja colocar de forma sobescrita deve ser efeito utilizando o elemento <sup>desse jeito o conteudo envolvido pelo elemento ficará sobescrito


Conteudo Subescrito

todo conteudo que se deseja colocar de forma subescrita deve ser efeito utilizando o elemento <sub>assim o conteudo envolvido ficara de forma subescrita
