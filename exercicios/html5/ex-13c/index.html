CSS Externo

O que é?

 Esse tipo de estilização se distingue por ser um recurso externo a HTML, ou seja CSS Externo é um arquivo independente, sua extensão é .css e é dentro desse arquivo onde ficaram todas as estilizações feitas e essas estilziações poderam ser vinculadas a arquivos .html através do elemento link="" este possuira os parametros rel="" href="" rel é como o que é o recurso que esta sendo relacionado ao arquivo, no caso o valor é "stylesheet" porque o recurso é um arquivo de folha de estilo. href você já sabe é onde declaramos a localização do recurso. um padrão é que se desenvolva um arquivo chamado style.css mas você além de poder nomear o que quiser (desde que esteja diacordo com a convenção de nomeatura) também pode vincular quantos arquivos .css quiser, basta desenvolver um elemento link para cada um e ir vinculando aos teus arquivos .html ASSIM A ESTILIZAÇÃO DO ARQUIVO .CSS SERA TRANSPORTADA PARA TODOS OS ARQUIVOS .HTML é muito mais simples e pratico usar esse metodo de estilização

Falando mais sobre CSS

 Sabia que tem como você criar regras para a sua folha de estilo CSS. sim @charset "" é uma regra que só pode ser criada dentro de um arquivo .css e como você deve ter percebido essa é uma regra relacionada a conjunto de caracteres, no caso ela vai garantir que o seu arquivo .css possua compatibilidade com acentuações. Essa regra não é obrigatoria mas caso você precise resolver problemas com acentuação coloque a regra antes de desenvolver qualquer coisa no arquivo .css
