Links Externos

 Link Externo é toda area clicavel que transporta o usuario para um endereço que esteja além do site onde ele esteja. como ir de um site para outro site exemplo.
 Se você interagiu com esse "texto em azul" meus parabêns você acabou de interagir com um Hiperlink que resumidamente é esse processo de ser transportado de um local para outro, especialmente com textos os Hiperlinks começaram desse jeito; Quer mais um exemplo Testa ai
 Ambos os links desse paragrafo são considerados Links Externos justamente porque eles possuem em seu parametro href="" uma URL ABSOLUTA declarada como valorMas o que seria uma URL absoluta? bem URL Absoluta é toda URL que não pertença a Documentação original de um "projeto", no caso da HTML5 seria todo endereço que não se localize no conjunto de arquivos e pastas que juntos formam a documentação da tal aplicação HTML5, que pode ser um site, um projetinho o que for. RESUMO:SE O ENDEREÇO QUE FOR DECLARADO COMO VALOR NO PARAMETRO href="" FOR "ESTRANGEIRO" A DOCUMENTAÇÃO DA SUA APLICAÇÃO ENTÃO ESTA URL É ABSOLUTA E ESTE É UM LINK EXTERNO Entendimento

Links para Download

 O desenvolvimento de links que possuam conteudos baixaveis é semelhante ao de links de neavegação o que muda são alguns parametros especificos para esse tipo de elemento veja exemplos abaixo:


Apos a formatação do seu arquivo pdf, seu download estará disponivel.
Caso queria seu arquivo compactado no formato .zip, seu Download está disponivel

 Acima nós temos dois exemplos de conteudo para download, o primeiro contem um arquivo, no caso em .pdf e a estrutura do elemento <a> é facil de entender; Seu parametro href="" possui como valor a URL do arquivo que será disponibilisado para download, que é seguido do parametro download, este é um parametro que indica que o conteudo do elemento <a> é disponivel para download, declarando o nome do arquivo como valor mas não tome isso como garantia porque não é todo Navegador que reconhece esse parametro mas para garantir que o conteudo de um elemento que tem a função de ser baixado o seja utilize o parametro type="" e declare como valor qual é a categoria do arquivo que o elemento esta contendo. No caso dos exemplos acima o primeiro possui "application/pdf" declarado como valor do parametro type, isso diz que o conteudo do elemento <a> esta disponibilisando é uma aplicação,(que é um arquivo) do formato .pdf já o segundo exemplo possui "application/zip" declarado como valor do seu parametro type="" o que vai garantir que esse formato de arquivo seja baixavel
