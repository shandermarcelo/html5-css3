Links de Patrocineo

 Também existe valores semânticos para você indicar para o Mecanismo de busca que um link é de patrocinador. Veja quais são e a função de cada valor agora.

Patrocinador
Patrocinado

 Antes da explicação entenda que a estrutura de um link patrocinado é indica a de um link externo ou seja vai ter URL Absoluta declarada no parametro href="" vai ter o idioma nativo declarado no paramteo hreflang="" vai ter declarado "_blank" no parametro target="" porém o que distingue um link patrocinado de um link externo é o valor declarado no parametro rel="" no caso de link patrocinado dois valores podem ser declarados nesse parametro "nofollow" é um valor que indica que o link é de patrocinador é que quando o mecanismo compreender isso ele não precisa continuar a interagir com o conteudo desse link porem nem todo patrocinador vai querer isso portanto o valor "dofollow" existe para que apos o mecanismo de busca compreender que o link patrocinado é de patrocinador ele deve interagir com seu conteudo. Esses é que são os possiveis valores a serem declarados no parametro rel="" de um link patrocinado
