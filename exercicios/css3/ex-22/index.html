Pseudo Classes

É considarada uma Pseudo Classe toda estilização auxiliar que visa manipular o estado de um elemento. Desenvolvidas através de seletores as Pseudo classes são conteudos que desenvolvemos quando queremos manipular "O que vai acontecer quando o elemento estiver em *** estado" veja alguns exemplos de como as pseudo classes funcionam

Este é o exemplo de pseudo classe que faz uso das estilizações para o estado hover este é o estado onde o conteudo possui o cursor sob si e o código de desenvolvimento é o seguinte

                    
                #hover:hover {
                    color: yellow;
                }

Como você pode ver o estilo de pseudo classe é desenvolvido através de um atrelamento ao seletor, que pode ser um ideitificador único ou uma classe ou diretamente no elemento. Pseudo Classes são representadas pelo simbolo de : você declara esse simbolo antes de desenvolver a pseudo classe.

Aqui é o exemplo de pseudo classe que usa estilos para quando o conteudo esta no estado de ativação, basicamente o que aconteçe quando interajem com o conteudo. a estrutura de desenvolvimento é a mesma do hover, eu fiz esse exemplo para demonstrar que existem muitas pseudo classes. a estrutura é a seguinte

            
                #active:active{
                    color: red;
                }

e ai está a estrutura

Afiliamento

Primeiramente Afiliamento é o nome dado a conteudos que se encontram dentro de outros conteudos. o elemento <html> "afilia" todos os conteudos desenvolvidos para uma aplicação HTML então você pode dizer que "toda a estrutura HTML é filha do elemento html porque todos os conteudos estão contidos dentro dele" e é justamente isso que é a tal afiliação. E você pode desenvolver estilos diretamente para esses "filhos" veja o exemplo

        
            div > p {
                font-family: 'Courier New', Courier, monospace;
                font-size: medium;
                font-weight: bold;
            }
    
            div > p:hover {
                font-family: 'Franklin Gothic Medium', 'Arial Narrow', Arial, sans-serif;
                font-size: large;
                font-weight: bolder;
            }
    
            div > p:active {
                font-family: 'Gill Sans', 'Gill Sans MT', Calibri, 'Trebuchet MS', sans-serif;
                font-size: x-large;
                font-weight: lighter;
            }

Viu os exemplos acima demonstram como eu estilizei os conteudos textuais dos exemplos de uso de pseudo classe. como o elemento <div> não possui suporte para estilizar conteudo textual eu estilizei eles através da afiliação, desenvolvendo elementos <p> dentro das divs, assim os afiliando a elas e ai eu fui estilizando diacordo com o proposito das pseudo classes que eu desenvolvi para cada div
