Pesudo Elementos

Pseudo Elementos são "meio elementos" que tem a função de colocar conteudo antes ou depois de um elemento, no caso dos estilos estes pseudo elementos vão permitir que se desenvolva estilos antes ou depois de um conteudo, este pode ser um conteudo HTML puro ou algo que possua estilos. Por ser algo que ou fica entre um conteudo ou apos este o uso dos pseudo elementos se limitam apenas a elementos que possuam componentes de abertura e fechamento, as famosas <>

Os valores que se pode declarar para pseudo elementos são

  • :: é o simbolo que representa os pseudo elementos, todos devem ser declarados apos este simbolo
  • ::before para desenvolver estilos antes do inicio do elemento
  • ::after para desenvolver estilos depois do termino do elemento
  • ::firstline para desenvolver estilos na primeira linha de um paragrafo
  • content é o que precede todo os estilos que se poça fazer em um pseudo elemento, geralmente seus valores são declarados entre ''
