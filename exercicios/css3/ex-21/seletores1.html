Seletores O que são

Seletores são elementos e estilos que tem a função de destacar um ou mais elementos afim de garantir que estilizações aplicadas sejam de exclusividade do(s) elemento(s) que possuem tal parametro.

Seletor id

O seletor id="" é o tipo de seletor que se usa quando a intenção é aplicar estilos a um único elemento. na HTML5 usamos o parametro id="" para darmos a um elemento seu ideitificador único, esse ideitificador será o valor declarado. Após isso na CSS3 fazemos a conexão declarando # que é acompanhada do valor declarado na HTML5. Assim os estilos desenvolvidos para o ideitificador seram aplicados somente a esse elemento que possui tal id. SABIA QUE É POSSIVEL FUNDIR ESTILOS DE UM ID COM O DE UMA CLASSE PORÉM OS ESTILOS DE ID TERAM SUPERIORIDADE HIERARQUICA AOS ESTILOS DA CLASSE é feito id="" class=""

Seletor class

O seletor class="" é o tipo de seletor que se usa quando a intenção é aplicar estilos a multiplos elementos, estes geralmente possui caracteristcas em comum e por isso estes são agrupados em um seletor class para que as estilizações somente a esse grupo de elementos. Resumo e conclusão

id="" é o seletor que se usa para destacar e estilizar um único elemento. Na CSS3 tanto o vinculo quando a estilização é feita através da # esta é acompanhada pelo valor declarado no parametro id, que é o que garante o vinculo. class="" é o seletor que se usa para destacar e estilizar multiplos elementos, estes geralmente possuem caracteristicas semelhantes e todos possuem o parametro class="" que terá declarado o valor, que é o componente vinculador, Na CSS3 usamos o . que é acompanhado do valor declarado no parametro class.
