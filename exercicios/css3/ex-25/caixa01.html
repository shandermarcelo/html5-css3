Conteudo e Box Size

Primeiramente todo conteudo desenvolvido fica dentro dessa caixa, e a primeira parte que notamos é a area onde o conteudo se encontra, no caso de caixas que possuem o display block essa area onde o conteudo fica se expande até o maximo da resolução de tela do dispositivo do usuário, mesmo que o conteudo em si não ocupe toda essa expensão, ela é demarcada por padrão devido a natureza do display block essa caracteristica não se aplica para elementos que possuem o display inline, aqui a area do conteudo será diacordo com o tamanho do mesmo.

Outra caracteristica são as medidas dessa area de conteudo width representa a medida de largura. height representa a medida de altura é a essas duas medidas damos o nome de box-size lembre que o tamanho da caixa não inclui medidas de padding,border,outline e margin. não inclui nenhuma outra medida além das de box-size

padding

Estilizações de padding devem ser feitas da seguinte forma

  1. padding-top
  2. padding-right
  3. padding-bottom
  4. padding-left

Sempre deve fazer esse movimento circular, da direita para a esquerda. Agora falando sobre padding estas são medidas que tem como função gerar um preenchimento interno, isso fara com que o conteudo seja movido para dentro da caixa, se afastando das suas bordas (caso elas sejam aplicadas) As medidas foram declaradas acima mas sabia que você pode desenvolver padding para partes especificas, caso queira

border

Essa é a medida que desenvolvemos depois da padding, border tem a função de gerar bordas para a caixa, os estilos básicos de border são os seguintes

        
            border-width: 10px;
            border-style: solid;
            border-color: black

Como você vê toda borda deve ter uma espessura, no caso width. Toda border deve ter uma forma, no caso style. E toda borda deve ter uma cor para essa forma, no caso black saiba que existe muitos tipos de estilos para bordas e o acima é só um exemplo

outline

Pouco utilizado outline é um traço que separa a borda da margin, com o calculo de sua espessura fazendo parte da margem estabelecida, veja um exemplo de estilos outline e note como essas medidas são identicas as de uma borda, Isso quer dizer que a área onde o outline sera exibido e se desenvolvera é dentro da área da margin

        
            outline-width: 5px;
            outline-style: dashed;
            outline-color: red;

margin

Margin é muito importante porque é através dela que podemos gerar espaço entre uma caixa e outra. assim como padding as medidas de margin são desenvolvidas de top,right,bottom,left. e é possivel centralizar caixas usando o estilo margin: auto; E da mesma forma que desenvolvesse padding, se desenvolve margin veja abaixo

        
            margin-top: 8px;
            margin-right: 8px;
            margin-bottom: 8px;
            margin_left: 8px;

shorthands

Como se sabe shorthands servem como uma forma de economizar linhas de código e agora vamos aprender alguns shorthands que podemos usar para estilos de elementos para nossa box model

Dica para calculo de elementos de caixa

para calcular o tamanho total de um elemento em caixa some as medidas do conteudo + medidas de preenchimento interno + medidas de borda + medidas de margin (não se inclue as medidas de outline, que é o contorno porque, como dito este estilo faz parte da)

Shorthand para Border
border: border-width border-style border-color
Shorthand para Padding
padding: padding-top padding-right padding-bottom padding-left
Shorthand para Outline
outline: outline-width outline-style outline-color
Shorthand para Margin
margin: margin-top margin-right margin-bottom margin-left

Simplificando o Simplificado

É possivel deixar até mesmo as declarações das shorthands box model mais simples, para isso aconteçer 1 condição deve ser atendida, e ela aconteçe quando os 4 valores (top right bottom left) formam 2 pares diferentes, isso quer dizer que se os valores top e bottom forem iguais, e os valores right left forem iguais mas entre os pares esses valores se diferem, então se pode declarar apenas esses valores 2 vezes. veja o exemplo

        
            padding: 10px 15px;

            margin: 20px 30px;

Lembresse que essa simplificação não pode ser aplicada para shorthands de border e outline, isso porque os estilos dessas declarações não são direcionaveis, como são os estilos de padding e margin portanto essa declaração acima pode ser entendida como (padding top=10px right=15px bottom=10px left=15px margin top=20px right=30px bottom=20px left=30px)

Valor único

Também é possivel declarar um valor único para estilos de padding e margin, a declaração fica asism

        
            padding: 10px;

            margin: 20px;

Acima está exemplos de como o valor único funciona. quando todos os estilos direcionais possuem o mesmo valor então se declara este apenas uma vez, assim o valor declarado vai se aplicar a todos os estilos seguintes

Margin e Centralização

Para centralizar uma box model se declara "auto" como valor mas quando se deseja centralizar as laterais então de repete essa declaração veja exemplos

        
            margin: auto;

Cnetralizando em Descrepancia

Usar um valor único para a shorthand margin pode não ser muito prático porque na maioria das vezes as propriedas top e bottom podem possuir valores diferentes e nesse caso a declaração completa deve ser feita veja

        
            margin: 30px auto 25px auto;

O exemplo acima demonstra o que fazemos em situações assim, mesmo uma shorthand sendo um elemento de multiplas propriedades se houver discrepancias declarações escpecificas para cada propriedade devem ser feitas
