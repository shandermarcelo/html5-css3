O exemplo acima demonstra como Pseudo Elementos podem ser desenvolvidos. basicamente a função de um pseudo elemento é de permitir que se desenvolva conteudos nas periferias de um elemento. (entenda isso como as laterais como afrente do elemento e atras do elemento) pseudo elementos são representados pelos dois pares de dois pontos :: e algumas das possiveis declarações são as seguintes

  • ::before Desenvolve conteudo antes ou afrente do elemento
  • ::after Desenvolve conteudo depois ou atras do elemento
  • ::first-letter Desenvolve conteudo referente a primeira letra de um conteudo textual
  • ::first-line Dsenvolve conteudo referente a primeira linha de um conteudo textual

Agora veja como é feita uma declaração de pseudo elementos

        
            a::before {
                content: '[Link] ';
    
            }
            a::after {
                content: ' [Link]';
            }

O código acima demonstra como é a estrutura de uma declaração de pseudo elementos. primeiro se vincula o pseudo elemento ao seletor que recebera os estilos e depois se desenvolve usando content, colocando o valor que sera o conteudo do pseudo elemento entre aspas
