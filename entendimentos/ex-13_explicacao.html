CSS Inline

 CSS é a linguagem de estilização que se tornou responsável pela forma do conteudo HTML apartir da sua quinta versão e nesse exercicio o método de implementação das estilizações utilizado foi o método CSS Inline inclusive as palavras "CSS Inline" acabaram de receber uma estilização, no caso a "color : blue;" que é o comando que mudar a coloração; Enfim o que você deve saber é que esse método de estilização é implementado diretamente no elemento através do parametro style="" este possuira como valor os comandos que vão manipular a forma, e saiba que a declaração de um estilização termina com os simoblos de ; caso queria mais estilizações e esteja utilizando o método CSS Inline separe as etilizações com um espaço em branco apos o fim do ; e desenvolva o codigo apos o espaço em branco gerado

Considerações

 Como você viu no exercicio 13 o uso de CSS Inline não deve ser exagerado porque muitos parametros style vão poluir o código e tornado dificil de compreender; Sem se esquecer que não é possivel escalar a implementação de uma estilização devido a especificidade da CSS Inline (CSS Inline esta no topo da hierarquia de estilização)mas é bom saber que existe esse método de implementação e que ela pode até ter seu uso pontual.
