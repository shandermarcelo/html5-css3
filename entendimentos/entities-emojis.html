HTML Entities

 O termo HTML Entities é usado para códigos que são usados para rerpesentar caracteres especiais e simbolos, um bom exemplo do uso disso são os colchetes angulares(simbolo de menor que e maior que)
 Imagine um desenvolver que quer representar os simbolos de Colchete Angular visualmente em formatdo textual para os usuarios, ele não poderá fazer isso apenas precionando as teclas que representão esses simbolos no seu teclado, isso porque a linguagem HTML já foi programada para reconhecer essas teclas como as ativadoras dos seus componentes de tags então como o desenvolvedor vai representar esses simbolos visualmente para os usuarios? Ele terá 2 opções, sendo uma delas trabalhosa e a outra simples vamos a primeira:
1ª O desenvolvedor vai criar uma imagem que possua os simbolos dos colchetes angulares e com a imagem pronta ele vai colocala no seu código através do elemento <img> através do parametro src="" declarando a URL da imagem como valor, e ai ele vai fazendo as declarações nos outros parametros desse elemento; Como você imagina a imagem vai ter que ser encaixar no texto, vai ser um trabalhão. Agora vamos para a segunda opção E ESSA É A OPÇÃO CERTA!
 O Desenvovlerdor vai utilizar HTML ENTITIES correspondentes a estes simbolos para representalos. O que são HTML ENTITIES? elas são um conjunto de códigos onde cada entity representa um simbolo próprio, veja uma lista com algumas entities:

  • &nbsp; representa um efeito de um espaço em branco, esse espaço é pequeno mas pode ser repetindo muitas vezes.

  • &lt; representa o simbolo de Menor que, o tal do colchete angular

  • &gt; representa o simbolo de Maior que, também é um colchete angular

  • &copy; representa o simbolo de copyright, ou os direitos autorais, o conteudo que for colocado grupado a introdução desse simbolo vai ser considerado as cradenciais do dono da copyright olha o exemplo ©Shander Marcelo Note que o nome é sobrenome ficaram com destaque

  • &trade; representa o simbolo de "autoria de conteudo", que é quando um idvividuo, empresa ou conglomerado possuem um simbolo, design ou expressão que faz com que seu produto ou marca tenham sua fonte ou origem relacionada a tal grupo, o que os distinguiras das demais marcas e produtos. Esse simbolo é usado para produtos os serviços; O que o diferencia do copyright

  • &reg; representa que o conteudo esta vinculado a uma marca registrada legalmente pelo governo, diferente o simbolo de ™ que não possui base juridica o simbolo de marca registrada possui e isso é uma das coisas que mais o distingue do nosso ™ ,isso também implica que o uso desse simbolo MUITO MAIS RESTRITO, o que significa que somente individuos ou empresas que possuem tal registro de marca no governo podem usar, esse registro geralmente dura por uma década e o uso desse simbolo sem o registro pode causa problemas judiciais reais para quem inflingir tal regra
