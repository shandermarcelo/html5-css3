CSS Interno (Local)

Caracteristicas

 Esse metodo de estiulização se distingue por causa da sua estrutura de estilização, CSS Interno tem como base estrutural o elemento <style>, este elemento tem como função ser o container de todas as estilizações do CSS Interno, basicamente atua como uma folha de estilização local, que é nativa da página onde ela esta. e isso também limita o uso desse método porque para casa página um elemento seria necessario, sem citar que o tamanho do conteudo que possui estilizações é muito grande e pros preguiçosos é chato ter que descer linhas de código.Enfim esse método é melhor que CSS Inline porque polui menos o código fonte mas não é melhor que o terceiro método, use ele pontualmente. (CSS Interno esta no meio da hierarquia de estilização)
